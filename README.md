📱 Projeto Scrunic — Plataforma de Gerenciamento Ágil

O Scrunic é uma aplicação desenvolvida para auxiliar no gerenciamento de projetos ágeis, permitindo organização de tarefas, acompanhamento de progresso e colaboração entre equipes.

O projeto foi criado com o objetivo de demonstrar boas práticas no desenvolvimento de aplicações web utilizando React no frontend e Node.js no backend, com integração a banco de dados e documentação de APIs.

✨ Descrição do Projeto

O Scrunic funciona como uma plataforma de apoio a metodologias ágeis, permitindo que usuários criem projetos, organizem tarefas e acompanhem o andamento das atividades em tempo real.

A aplicação simula o fluxo de um sistema completo de gestão ágil, com backend estruturado, banco de dados e documentação de endpoints.

O projeto é ideal para demonstrar:

Estrutura de aplicações fullstack (frontend + backend);

Organização de projetos ágeis (tasks, status, progresso);

Integração com banco de dados relacional;

Documentação de APIs com Swagger.

🧩 Funcionalidades

📋 Gerenciamento de Projetos:
Criação e organização de projetos ágeis.

🗂️ Controle de Tarefas:
Adição, edição e acompanhamento de tarefas (ex: backlog, em andamento, concluído).

📊 Acompanhamento de Progresso:
Visualização do status das atividades dentro do projeto.

🔗 API Documentada:
Endpoints documentados utilizando Swagger para testes e integração.

🧪 Testes de Banco de Dados:
Estrutura preparada para validação e testes das operações com o banco MySQL.
